Change input file type appearence

```html
<x-administrable::custominputfile
    selector=".simple-file-input"
    width="100%"
    placeholder="Choisir une image"
    buttonText="Choisir"
    allowedExts="['png', 'gif', 'jpg', 'jpeg']"
/>
```
